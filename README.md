# 🐱 CatchMeow (캐치냥)

마을의 개성 넘치는 고양이들을 카메라로 찍어 컬렉션을 완성하세요!

## 🎮 게임 소개

**CatchMeow**는 고양이 수집 게임입니다. 플레이어는 카메라를 들고 마을을 돌아다니며 다양한 고양이들을 찾아 사진을 찍습니다. 각 고양이를 성공적으로 촬영하면 컬렉션에 저장되고, 모든 고양이를 모으면 다음 시즌으로 넘어갑니다.

## ✨ 핵심 특징

- **개성 있는 고양이들**: 각 고양이는 고유한 성격과 행동 패턴을 가지고 있습니다.
- **전략적 접근**: 고양이의 개성을 파악하고, 그에 맞는 방법으로 접근해야 사진을 찍을 수 있습니다.
- **컬렉션 시스템**: 촬영한 고양이들은 도감에 저장됩니다.
- **시즌제**: 모든 고양이를 모으면 새로운 시즌이 열립니다.
- **개인 플레이**: 나만의 속도로 고양이 컬렉션을 완성하세요.
